# 🎟️ EventXperience

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

> A full-stack event discovery and club management platform

Users can create and manage clubs, host events, and follow their favourite organizations to stay notified about upcoming plans. Whether you're organizing a local meetup or a large-scale conference, EventXperience has you covered.

## ✨ Features

- 🏛️ **Club Management** — Create and manage clubs with custom profiles
- 📅 **Event Creation** — Host events under your clubs with full detail pages
- 🎟️ **Event Registration** — Users can browse and register for events
- 🔔 **Follow & Notifications** — Follow clubs to get notified about upcoming events
- 🔐 **Authentication** — Secure user accounts and role-based access

### 🗺️ Roadmap (Post-MVP)

- 🚨 Report system for clubs and events
- 🏟️ Venue booking integration
- 🤖 AI chatbot to assist in event discovery and search
- 🌐 Web scrapers for external event aggregation

## ⚙️ Getting Started

### Prerequisites

- Node.js & Angular CLI
- .NET 8 SDK
- MySQL & Redis
- RabbitMQ

### Installation

```bash
# Clone the repository
git clone https://github.com/thomastran117/eventxperience.git
cd eventxperience

# Install frontend dependencies
cd client
npm install

# Run the frontend
ng serve

# Run the backend
cd ../server
dotnet run
```

> See the `docs/` folder for environment variable setup and full configuration instructions.
